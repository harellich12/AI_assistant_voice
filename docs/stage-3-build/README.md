# Stage 3: Build

This folder contains implementation execution artifacts, ready for engineering delivery tooling.

## Files
1. `build-plan.md`: Sprint-by-sprint sequencing, dependencies, and critical path.
2. `tickets-stage-3.csv`: Import-ready ticket breakdown for Jira/Linear mapping.
3. `import-mapping.md`: Field mapping guidance for Jira and Linear.
4. `jira-AAA-import.csv`: Jira-ready import file targeted to project key `AAA`.
5. `jira-AAA-traceability.csv`: Document-to-ticket mapping matrix.
6. `jira-AAA-board-guide.md`: Import instructions for board `AAA/boards/3`.
7. `jira-automation.md`: Local script workflow for automatic Jira transitions from commits.

Translate Stage 2 execution design into actionable, trackable engineering work items.
